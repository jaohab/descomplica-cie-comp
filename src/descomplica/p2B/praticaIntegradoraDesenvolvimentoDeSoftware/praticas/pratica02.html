Linguagens de programação

Java

Linguagem de programação de alto nível usada para desenvolver aplicativos em diversos sistemas operacionais. É uma linguagem orientada a objetos e pode ser usada em diferentes tipos de projetos, como aplicativos para desktop, jogos, aplicativos web, sistemas embarcados e dispositivos móveis.

JavaScript

Linguagem de programação interpretada e de alto nível que é usada principalmente para criar páginas web interativas e dinâmicas. É executada no lado do cliente, ou seja, no navegador do usuário, e pode ser usada para criar efeitos visuais, validar formulários, interagir com APIs e muito mais.
